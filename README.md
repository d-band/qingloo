Qingloo
===

## Install

```bash
# Install postgres

$ brew update & brew install postgres
$ postgres -D /usr/local/var/postgres
$ createuser qingloo
$ createdb db_qingloo

# Run app:

$ npm i --registry=http://registry.npm.taobao.org
$ node app.js
```
